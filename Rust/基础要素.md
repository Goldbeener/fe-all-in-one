## 数据类型

+ `数值`
	+ 整数
		+ 有符号整数
			+ i8、i16、i32、i64、i128、isize
		+ 无符号整数
			+ u8、u16、u32、u64、u128、uszie
	+ 浮点数
		+ f32 32位浮点数，单精度
		+ f64 64位浮点数，双精度（默认）
	 

**不同数字类型区别是什么？**
**各自的应用场景**

+ `字符串`
	+ &str 字符串切片 字面量
	+ String  堆上可变字符串

**两个字符串类型的区别，应用场景？**

+ `数组`
	+ [type; n] 固定长度n的数组
	+ vec![1，2] 动态长度数组 vector

+ `结构体`

struct objName { x: i32 }
