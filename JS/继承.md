
```js
const person = {
	isHuman: true,
	name: 'haha',
	get alias () {
		return this.name
	}
}

// 以下两种写法是等效的
const jame = {
	__proto__: person,
	name: 'James'
}
// jame { name, alias }

const jane = Object.create(person, {
	name: { value: 'jane' }
})
// jane { name, alias }
```

**值得注意的是**

这里原型对象中的getter，会在继承对象里面自动生成一个属性
